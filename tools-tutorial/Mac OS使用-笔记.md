# Mac OS使用-笔记

- 跳转
  - `command + <- / ->`

- 截图
  - `command+shift+4`

- 锁屏
  - `Ctrl + Command + Q`
- 软件包管理工具 brew
  - 搜索软件：`brew search mysql`
  - 安装： `brew install mysql@5.7`
  - 启动服务：`brew services start mysql@5.7`

- 资源搜索工具：Alfred
- 终端zsh / oh-my-zsh
  - `command + T` 新标签页
  - `command + N` 新窗口
  - `Shfit + Home` 跳到行首
  - `Shfit + End` 跳到行尾
  - `Alt + <- / ->` 按词组跳
