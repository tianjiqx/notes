# IDEA Rust插件使用

快捷键：

- **Ctrl+Alt+L** - 格式化代码，个人对rust文件新增配置'/' + 'F' ，因为与Ubuntu锁屏冲突（注意配置为使用rustfmt进行格式化，而非IDE内置的格式化工具）
- **Alt + F7** - Find Usage查找符号使用
- **Ctrl + 鼠标左键/Ctrl + B** - 跳转到定义 
- **Ctrl + Alt+B** - 跳转到实现，显示 trait 元素（方法、常量和关联类型）的实现列表。
- **Ctrl + Alt + <- / ->**  -  回退/前进 位置
  - 个人增加配置 **'`' + <- / ->**
