
# SkipList

空间换时间

| Operation Name    | Time Complexity | Space Complexity |
| ----------------- | --------------- | ---------------- |
| find-key(k)       | O(lgn)          | O(1)             |
| iter()            | O(n)            | O(n)             |
| insert(x)         | O(lgn)          | O(1)             |
| delete-key(k)     | O(lgn)          | O(1)             |
| delete-min/max()  | O(lgn)          | O(1)             |
| find-next/prev(k) | O(lgn)          | O(1)             |
| find-min/max()    | O(lgn)          | O(1)             |
| order-iter()      | O(n)            | O(1)             |
